    Tutorial

    Set up a practical drill in under a minute

    Place phones where the athlete can safely see and touch them. Pick a preset or custom drill, then start and track results.

    Tap zone

    1. Position phones

    Use a stable surface, mat, wall mount, or hand-held position. Keep taps quick but safe.

    Preset

    2. Pick a game

    Default presets enable leaderboards. Custom setups are best for practice variations.

    321

    3. Start countdown

    Start gives every connected device a 3-second cue before the clock begins.

    4. React

    Tap the lit screen. In Focus mode, hold when a distractor color appears.

    HostJoinPlay

    5. Add pods

    Host a room and send the join link. Connected phones use the profile name as their device ID.

    SaveRestRepeat

    6. Review results

    Rest countdowns prepare the next cycle. Results show the configuration and analytics for each attempt.

    Features

    What each mode is for

    Random Reaction

    Random color prompts for general reaction speed.

    Focus

    Tap only the target color and avoid distractors.

    Sequence

    Repeatable color sequences for memory and rhythm.

    Home Base

    Return-to-center drills using a home color between tasks.

    Default presets

    Standardized games that create fair leaderboards.

    Custom presets

    Save, edit, delete, and reload your own drill setups.
