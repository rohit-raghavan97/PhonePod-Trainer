      Tutorial

      Set up a practical drill in under a minute

      Place the phone where the athlete can safely see and touch it. Keep the screen awake, choose the drill rules, then start and record reactions.

      Tap zone

      1. Position the phone

      Use a stable surface, mat, wall mount, or hand-held position. Leave enough space so taps are quick but safe.

      Mode

      2. Choose the activity

      Start with Random Reaction for general speed, Focus for inhibition, Sequence for memory, or Home Base for return-to-center work.

      45s 20 hits 2.5s light

      3. Set timing rules

      Pick a time limit or hit target. Use light timeout to force faster reactions, and add delay when you want less predictable starts.

      4. Select colors

      Use fewer colors for beginners. For Focus mode, set one target color and treat the others as no-tap distractors.

      Start Tap Result

      5. Run the drill

      Press Start, tap only when the active target appears, then check hits, misses, false hits, and average reaction time.

      Save Rest Repeat

      6. Repeat consistently

      Save a preset when the setup works. Use cycles and rest seconds to compare multiple rounds fairly.

      Host Join Place pods

      7. Add phones as pods

      Open Multi, host a room, share the join link, then place connected phones around the drill area.

      Features

      What each control is for

      Random Reaction

      Shows random colors from your selected set. Best for simple reaction speed and warmups.

      Focus

      Only the target color should be tapped. Distractor colors train restraint, attention, and decision speed.

      Sequence

      Builds a repeatable color sequence for the round. Useful for memory, rhythm, and planned movement patterns.

      Home Base

      Alternates between a task color and a home color. Use it when the athlete must return to a start position after each action.

      Duration

      End by timeout, hit target, or whichever happens first. Time is better for conditioning; hit target is better for consistency.

      Lights out

      Choose whether a light disappears after a tap, after a timeout, or either. Timeout mode makes missed reactions visible.

      Delay

      Add no delay, fixed delay, or random delay before the next color. Random delay reduces guessing.

      Cycles and rest

      Run repeated rounds with planned breaks. This is the easiest way to compare performance while fatigue builds.

      Results

      Track hits, misses, false hits, best reaction, average reaction, history, and exportable CSV results.

      Multi-device rooms

      Host a room and connect other phones as pod screens. The host runs the drill while joined phones light up and send taps back.

      Use cases

      Example drills to try

      Solo reaction test

      Random Reaction, 30 seconds, light timeout 1.5s. Place the phone on a table and tap as fast as possible.

      Goalkeeper hand speed

      Random Reaction, 45 seconds, random delay. Mount the phone at shoulder height and tap with the called hand.

      Basketball closeout reset

      Home Base, 60 seconds, green as home. Touch the phone, retreat to a cone, then return when home appears.

      Focus and no-tap control

      Focus, target green, three distractors, three strikes. Tap only green and count false hits as failed reps.

      Rehab range-of-motion reps

      Hit target, 15 hits, longer light timeout. Put the phone at a safe reach angle and track controlled touches.

      Team circuit station

      Cycles set to 4, rest 20 seconds. Each athlete takes one cycle, then compare average reaction time.

      Multi-phone agility course

      Host on one phone and place joined phones around a small area. Athletes move to whichever phone lights up next.
